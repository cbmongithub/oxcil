# Oxcil

Oxcil is a concept for AI-assisted lead infrastructure and workflow experiences for local service businesses.

The project is focused on exploring how modern web tooling, structured workflows, and simple automation can help surface, organize, and route valuable work more effectively.

## What This Repo Is

- a public-facing concept and landing page base
- a place to explore product direction
- a sandbox for workflow and interface ideas
- a starting point for future iterations

## General Direction

Oxcil is intended to stay flexible.

Rather than locking into one narrow implementation too early, the project is being shaped around:

- clear product positioning
- reusable structure
- strong visual presentation
- room for future experimentation

## Current Focus

The current emphasis is on:

- setting up a clean foundation
- defining the visual system
- keeping the interface cohesive
- leaving space for future product work

## Notes

- This repository is a work in progress.
- Details may change as the project evolves.
- The private planning notes contain the deeper product direction.
